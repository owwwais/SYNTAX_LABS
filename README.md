# SYNTAX_LABS

A premium, high-fidelity web application built with Next.js, featuring advanced animations and a "Terminal Velocity" / "Crystalline Compute" aesthetic.

## Features

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: Tailwind CSS & Modern CSS
- **Animations**: GSAP & Framer Motion for cinematic experiences
- **Typography**: Optimized Geist font family

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Design Identity

SYNTAX LABS embodies a futuristic, high-performance aesthetic. The interface is designed to be responsive, interactive, and visually stunning, sticking to a strict design system of obsidian depths and crystalline highlights.

## Deployment

The easiest way to deploy is using the [Vercel Platform](https://vercel.com/new).
